## 💡 方案分级体系

### 基础方案
- **价格范围**: ¥2,000 - ¥8,000
- **疗程次数**: 2-6次
- **治疗时长**: 1-3个月
- **适用人群**: 轻度问题、预算有限、初次尝试

### 标准方案
- **价格范围**: ¥5,000 - ¥25,000
- **疗程次数**: 4-12次
- **治疗时长**: 3-6个月
- **适用人群**: 中度问题、追求稳定效果

### 高级方案
- **价格范围**: ¥10,000 - ¥80,000
- **疗程次数**: 6-18次
- **治疗时长**: 6-12个月
- **适用人群**: 重度问题、追求极致效果

## 🔧 快速查询工具使用

### 命令行查询
```bash
cd /root/claude/cqbcc-service/app/data
python3 query_needs.py 美白      # 查询美白相关方案
python3 query_needs.py 祛斑      # 查询祛斑相关方案
python3 query_needs.py 抗衰      # 查询抗衰相关方案
```

### Python API调用
```python
import json

# 加载需求映射
with open('needs-mapping.json', 'r', encoding='utf-8') as f:
    needs_data = json.load(f)

# 查询特定需求
need = next(n for n in needs_data['needs'] if n['id'] == 'whitening')
print(f"需求: {need['name']}")
print(f"推荐项目: {[p['project_name'] for p in need['recommended_projects']]}")
```
